# AMI Corpus Data Conversion

## Getting Started

1. Install Python 3:

      Go to [install Python 3](https://www.python.org/downloads/)

2. Install Required Python Packages:

    ```
    $ pip install -r requirements.txt
    ```

3. Download Dataset

    Estimated Size: 1.94 GiB.

    ```
    $ python download_dataset.py
    ```

</br>
